Assignments:

Assignment 1 - Basic HTML

This assignment consisted of us configuring our website to show the name of the class, our names, a class description, and a list of our assignments.

Assignment 2 - Basic CSS

In this assignment, we begin to style our main page and the assignment 2 page using CSS. We modified colors, fonts, background, etc. using CSS.

Assignment 3 - Page Layout

In this assignment we began to explore how CSS can be used to change the layout of the page. We also explored Flex Boxes.

Assignment 5 - Recreate CSS Page

This assignment required us to recreate a webpage that was given to us. We had to try and match the same styling elements from the original page.

Assignment 6 - JavaScript, Buttons, Functions, and more.

This assignment required us to begin implementing buttons and functions using javascript, css, and html.

Assignment 7 - Conditionals

This assignment consisted of creating 2 pages using one index.html file. The page contained two exercises (an age comparison tool and a donation goal chart). We then used javascript to make the 2 exercises work and to show only one exercise at a time.

Assignment 8 - Loops

This assignment involved making a picture of a man walking/running switch back and fourth while moving across it's div box. It also made a fundraising thermometer graphic fill based on user input. Intervals in JS were used to make both these functionalities work.

Assignment 9 - Arrays

In this assignment, we were tasked with building a site with two activities, both of which utilized arrays. The first activity changes quotes every 2 seconds from an array of quotes, and the second activity draws a rainbow row-by-row (every second) and places a pot-of-gold at the end of it.

Assignment 10 - Classes

In this assignment, we were tasked with building a website that would display toys. The toy objects must be created and displayed using javascript only. As you navigate the website, if you hover over a toy, its information will be displayed.

Assignment 11 - Parsing JSON

In this assignment, we were tasked with creating a website that would get its information by parsing through a JSON file. The information on this site was a list of movies with different bits of information about them and it was all gathered from a JSON file.

Assignment 12 - Server Side Hello World

This assignment tasked us with creating a hello-world website using Node.js. We created a javascript file, pushed it to github, and hosted it on render.

Project Website - Part 1

This assignment involved the creation of the project website using the wireframe. This was a mockup of the site with placeholder images. There was a large focus on format and less of a focus on color/decoration.

This assignment tasked us with developing our website further by replacing all placeholder images with actual images and adding color to the site. We were also instructed to ensure that all of our flexing properties were working correctly.

Project Website - Part 4

This assignment tasked us with developing our website further by adding form validation to the form that was already present on the site, creating a working 'contact me' form that sends emails to my email, and adding an iframe to the site (my iframes can be found in the activities and attractions box when you click on a section).

Project Website - Part 5

In this assignment, we were tasked with further developing our website so that the information about the activities and attractions in Delaware were gathered entirely from a JSON file. This allowed the form to submit new activities and attractions to the website to operate successfully.
