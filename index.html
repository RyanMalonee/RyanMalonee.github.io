Assignments:

Assignment 1 - Basic HTML

This assignment consisted of us configuring our website to show the name of the class, our names, a class description, and a list of our assignments.

Assignment 2 - Basic CSS

In this assignment, we begin to style our main page and the assignment 2 page using CSS. We modified colors, fonts, background, etc. using CSS.

Assignment 3 - Page Layout

In this assignment we began to explore how CSS can be used to change the layout of the page. We also explored Flex Boxes.

Assignment 5 - Recreate CSS Page

This assignment required us to recreate a webpage that was given to us. We had to try and match the same styling elements from the original page.

Assignment 6 - JavaScript, Buttons, Functions, and more.

This assignment required us to begin implementing buttons and functions using javascript, css, and html.

Assignment 7 - Conditionals

This assignment consisted of creating 2 pages using one index.html file. The page contained two exercises (an age comparison tool and a donation goal chart). We then used javascript to make the 2 exercises work and to show only one exercise at a time.
